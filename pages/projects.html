PacMan Game

This version of PacMan is written in HTML, CSS and Javascript to create a PacMan character that moves across the screen and collects dots that are randomly placed. The code detects collisions between PacMan and the dots, and removes the dots when a collision occurs. The game starts when the "start-button" is clicked.

Eye Exercise

This code demonstrates a simple mouse tracking effect on a webpage. It moves two HTML elements in response to the movement of the mouse on the page. As the mouse moves, the position of the balls changes to follow it, creating a visual effect where it appears as if the balls are following the mouse.
